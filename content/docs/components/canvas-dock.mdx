## Props

### CanvasDock

| Prop              | Type                                                                                                      | Description                           | Default  |
| ----------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------- | -------- |
| position          | "bottom" \| "left" \| "right"                                                                             | Position of the dock                  | "bottom" |
| collapsed         | boolean                                                                                                   | Whether the dock is collapsed         | false    |
| onCollapsedChange | (collapsed: boolean) => void                                                                              | Callback when collapsed state changes | -        |
| variant           | "wooden" \| "metallic" \| "modern" \| "glass" \| "neon" \| "watercolor" \| "chalk" \| "paper" \| "canvas" | Visual style of the dock              | "wooden" |
| className         | string                                                                                                    | Additional CSS classes                | -        |
| children          | React.ReactNode                                                                                           | Dock items                            | -        |

### CanvasDockItem

| Prop      | Type                                       | Description                  | Default |
| --------- | ------------------------------------------ | ---------------------------- | ------- |
| icon      | React.ReactNode                            | Icon element                 | -       |
| label     | string                                     | Tooltip label                | -       |
| selected  | boolean                                    | Whether the item is selected | false   |
| onClick   | () => void                                 | Click handler                | -       |
| variant   | "brush" \| "pencil" \| "palette" \| "tool" | Animation style              | "tool"  |
| className | string                                     | Additional CSS classes       | -       |
| children  | React.ReactNode                            | Alternative to icon prop     | -       |

## Examples

### Basic Usage

```tsx
import { CanvasDock, CanvasDockItem } from "@/components/ui/canvas-dock";

export default function BasicDock() {
  const [selectedTool, setSelectedTool] = React.useState<string | null>(null);

  return (
    <CanvasDock>
      <CanvasDockItem
        variant="brush"
        selected={selectedTool === "brush"}
        onClick={() => setSelectedTool("brush")}
        label="Brush"
      >
        <BrushIcon />
      </CanvasDockItem>
      <CanvasDockItem
        variant="pencil"
        selected={selectedTool === "pencil"}
        onClick={() => setSelectedTool("pencil")}
        label="Pencil"
      >
        <PencilIcon />
      </CanvasDockItem>
    </CanvasDock>
  );
}
```

### Controlled Collapse State

```tsx
export default function CollapsibleDock() {
  const [collapsed, setCollapsed] = React.useState(false);

  return (
    <CanvasDock collapsed={collapsed} onCollapsedChange={setCollapsed}>
      {/* Items */}
    </CanvasDock>
  );
}
```


### With Paint Effects

The dock items automatically show paint splatter effects when selected. The colors are randomized from a predefined palette, but you can customize this by modifying the `paintSplatterColors` array in the component.

```tsx
const paintSplatterColors = [
  "rgb(239 68 68)", // red
  "rgb(34 197 94)", // green
  "rgb(59 130 246)", // blue
  "rgb(168 85 247)", // purple
  "rgb(249 115 22)", // orange
];
```
