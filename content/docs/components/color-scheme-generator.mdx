## Features

The Color Scheme Generator is a versatile tool that helps you create harmonious color combinations:

### Color Selection
- Interactive color picker with real-time preview
- HSL sliders for precise color adjustments
- Direct hex code input support

### Harmony Modes
- **Complementary**: Two colors opposite each other on the color wheel
- **Analogous**: Three colors next to each other on the color wheel
- **Triadic**: Three colors evenly spaced around the color wheel
- **Tetradic**: Four colors evenly spaced around the color wheel
- **Monochromatic**: Different shades and tints of the same color

### Export Options
- One-click color copying to clipboard
- Export complete color schemes as JSON
- Randomize feature for inspiration
