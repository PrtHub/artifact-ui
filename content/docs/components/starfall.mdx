## Usage

```tsx
import Starfall from "@/components/ui/starfall";

export default function Example() {
  return (
    <div className="h-[400px]">
      <Starfall variant="cosmic" count={20} speed={1.5} />
    </div>
  );
}
```

## Variants

```tsx
<Starfall variant="aurora" />
<Starfall variant="neon" />
<Starfall variant="ocean" />
<Starfall variant="cosmic" />
<Starfall variant="minimal" />
<Starfall variant="gradient" />
<Starfall variant="sunset" />
<Starfall variant="forest" />
```

## Background Effects

You can customize the background effects using these props:

```tsx
<Starfall
  variant="cosmic"
  showBackground={true} // Toggle background effects
  backgroundStars={30} // Number of background stars
  backgroundOpacity={0.3} // Background star opacity
/>
```

## Custom Backgrounds

The component works great with custom backgrounds:

```tsx
<div className="relative h-[400px]">
  {/* Custom background image */}
  <Image src="/your-image.jpg" alt="Background" fill className="object-cover" />

  {/* Starfall overlay */}
  <Starfall variant="minimal" showBackground={false} count={20} speed={1} />
</div>
```

## Props

| Prop                | Type                                                                                           | Default    | Description                                       |
| ------------------- | ---------------------------------------------------------------------------------------------- | ---------- | ------------------------------------------------- |
| `variant`           | `"cosmic" \| "aurora" \| "minimal" \| "gradient" \| "neon" \| "sunset" \| "ocean" \| "forest"` | `"cosmic"` | The visual theme of the starfall effect.          |
| `count`             | `number`                                                                                       | `25`       | Number of falling stars to display.               |
| `speed`             | `number`                                                                                       | `1`        | Animation speed multiplier for the falling stars. |
| `showBackground`    | `boolean`                                                                                      | `true`     | Toggle the themed background and effects.         |
| `backgroundStars`   | `number`                                                                                       | `30`       | Number of static background stars to display.     |
| `backgroundOpacity` | `number`                                                                                       | `0.3`      | Opacity of the background stars (0-1).            |
| `className`         | `string`                                                                                       | -          | Additional CSS classes to apply to the container. |
