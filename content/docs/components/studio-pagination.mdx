## Basic Usage

```tsx
"use client";

import { StudioPagination } from "@/components/ui/studio-pagination";

export default function PaginationExample() {
  const [currentPage, setCurrentPage] = React.useState(1);

  return (
    <StudioPagination
      totalPages={10}
      currentPage={currentPage}
      onPageChange={setCurrentPage}
      variant="default"
      animation="fade"
      showProgress
    />
  );
}
```

## Props

| Prop             | Type                   | Default   | Description                           |
| ---------------- | ---------------------- | --------- | ------------------------------------- |
| totalPages       | number                 | Required  | Total number of pages                 |
| currentPage      | number                 | Required  | Current active page                   |
| onPageChange     | (page: number) => void | Required  | Callback when page changes            |
| variant          | PaginationVariant      | "default" | Visual style variant                  |
| shape            | PaginationShape        | "rounded" | Button shape style                    |
| size             | PaginationSize         | "md"      | Component size                        |
| animation        | PaginationAnimation    | "fade"    | Animation style for transitions       |
| showProgress     | boolean                | false     | Show progress bar                     |
| enableKeyboard   | boolean                | true      | Enable keyboard navigation            |
| enableSwipe      | boolean                | true      | Enable touch swipe navigation         |
| enableMouseWheel | boolean                | true      | Enable mouse wheel navigation         |
| showFirstLast    | boolean                | true      | Show/hide first and last page buttons |
| isLoading        | boolean                | false     | Loading state                         |
| className        | string                 | -         | Additional CSS classes                |
| customColors     | CustomColors           | -         | Custom color overrides                |

## Accessibility

The component includes full keyboard navigation support and proper ARIA labels:

- Left/Right arrow keys for navigation
- Touch swipe support for mobile devices
- Mouse wheel navigation
- Loading state announcements
- Proper focus management
- ARIA labels for all interactive elements

## Styling

The component uses Tailwind CSS for styling and supports both light and dark modes out of the box. All styles can be customized using the `className` prop or the `customColors` prop for specific color overrides.

### Dark Mode Support

The component automatically adapts to dark mode when the `dark` class is present in the parent tree. It includes proper color adjustments for background, text, and hover states.

## Notes

- The component requires the `"use client"` directive as it uses client-side interactivity
- Animations are powered by Framer Motion
- Touch interactions use react-swipeable
- Keyboard navigation uses react-hotkeys-hook
