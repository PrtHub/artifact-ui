## Usage

```tsx

const images = [
   // Add your image URLs or objects here
]


export default const Example = () => {
   return <BlurGallery images={images} />

```

## Props

| Prop        | Type           | Default | Description            |
| ----------- | -------------- | ------- | ---------------------- |
| `images`    | `ImageProps[]` | `[]`    | Array of image objects |
