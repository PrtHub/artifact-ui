## Installation

<Steps>

<Step>Copy and paste the following code into your project.</Step>

`components/ui-canvas/grid-pattern-background.tsx`

<ComponentSource name="grid-pattern-background" />

<Step>Add the following code to your `tailwind.config.cjs` file:</Step>

```tsx
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    // your existing code
  ],
  theme: {
    extend: {
      // your existing code
      animation: {
        "dots-shift": "dots-shift 20s linear infinite",
        "lines-shift": "lines-shift 20s linear infinite",
      },
      keyframes: {
        "dots-shift": {
          "0%, 100%": {
            transform: "rotate(0deg)",
          },
          "50%": {
            transform: "rotate(180deg)",
          },
        },
        "lines-shift": {
          "0%": {
            transform: "translateX(0) translateY(0)",
          },
          "50%": {
            transform: "translateX(5px) translateY(5px)",
          },
          "100%": {
            transform: "translateX(0) translateY(0)",
          },
        },
      },
    },
  },
};
```

<Step>Update the import paths to match your project setup.</Step>

```tsx
import GridPatternBackground from "@/components/ui-canvas/grid-pattern-background";
```

</Steps>

## Usage

```tsx
import GridPatternBackground from "@/components/ui-canvas/grid-pattern-background";

export default function Example() {
  return (
    <div className="relative h-[200px]">
      <GridPatternBackground
        gridType="dots"
        gridSize={24}
        opacity={0.3}
        color="#6d28d9"
        animate={true}
      />
      <div className="absolute inset-0">Your content here</div>
    </div>
  );
}
```

## Props

| Prop      | Type              | Default        | Description                      |
| --------- | ----------------- | -------------- | -------------------------------- |
| gridType  | 'dots' \| 'lines' | 'dots'         | Type of grid pattern to display  |
| gridSize  | number            | 20             | Size of the grid cells in pixels |
| opacity   | number            | 0.2            | Opacity of the grid pattern      |
| color     | string            | 'currentColor' | Color of the grid pattern        |
| animate   | boolean           | false          | Enable subtle animation          |
| className | string            | ""             | Additional CSS classes           |
