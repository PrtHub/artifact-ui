## Installation

<Steps>

<Step>Copy and paste the following code into your project.</Step>

`components/ui/artistic-navigation.tsx`

<ComponentSource name="artistic-navigation" />

<Step>Update the import paths to match your project setup.</Step>

```tsx
import { ArtisticNavigation } from "@/components/ui/artistic-navigation";
```

</Steps>

## Usage

```tsx
import { ArtisticNavigation } from "@/components/ui/artistic-navigation";

export default function Example() {
  const navItems = [
    { label: "Home", href: "#", icon: <HomeIcon /> },
    { label: "Profile", href: "#", icon: <UserIcon /> },
    // Add more items as needed
  ];

  return <ArtisticNavigation items={navItems} />;
}
```

## Props

| Prop name | Type                       | Default  | Description               |
| --------- | -------------------------- | -------- | ------------------------- |
| items     | `ArtisticNavigationItem[]` | Required | Array of navigation items |
| className | `string`                   | ''       | Additional CSS classes    |
